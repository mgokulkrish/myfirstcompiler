# MyFirstCompiler
First compiler based on the LLVM Kaleidoscope with JIT support.

# Run command
```
clang++ -g main.cpp `llvm-config --cxxflags --ldflags --system-libs --libs core orcjit native` -O3 -std=c++17 -o main
```
